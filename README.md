deploying grafana in docker-compose
### 1. create empty grafana.ini
### 2. start container:
 ``` bash
docker-compose up -d
 ```
### 3. open in browser:  
 ``` bash
http://localhost:3000 
```
### 4. stop container:  
 ``` bash
docker-compose down
```
